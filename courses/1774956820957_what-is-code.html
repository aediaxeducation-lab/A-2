    # Meal Pass Management System - Software Requirements Specification (SRS)

This repository contains the complete **Software Requirements Specification (SRS)** for the **Meal Pass Management System** designed for **SAPL Industries Pvt Ltd**. The document follows standard SRS structure and is presented in a clean, iOS-inspired webpage format.

## 📋 Project Overview

The Meal Pass Management System is designed to manage employee meal access through barcode-enabled cards at SAPL Industries, which has:
- **150 Staff Members**
- **2000 Workers**

The system tracks daily meal consumption, prevents duplicate meal usage, and integrates with attendance and payroll deduction systems.

## 🚀 Key Features

- **Employee Management**: Register and manage employees with barcode ID assignment
- **Barcode Card Integration**: Unique PVC cards with employee barcodes
- **Meal Tracking**: Record breakfast and lunch consumption
- **Smart Rules**: Prevent duplicate meals (max 1 breakfast + 1 lunch per day)
- **Payroll Integration**: Automatic deduction calculation based on meal costs
- **Reporting System**: Export reports in Excel, CSV, and PDF formats
- **Offline Operation**: Local database with cloud synchronization

## 🛠 Technology Stack

| Component | Technology |

| Backend | Node.js |
