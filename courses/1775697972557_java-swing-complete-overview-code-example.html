    Java Swing

    📘 Java Swing is a lightweight, platform-independent, GUI (Graphical User Interface) toolkit for Java. It is built on top of Abstract Window Toolkit (AWT) and provides a rich set of components such as buttons, tables, trees, text areas, and dialogs. Swing is written entirely in Java, which makes it highly portable and customizable with pluggable look‑and‑feel (PLAF). Swing components follow the Model‑View‑Controller (MVC) architecture and are part of the javax.swing package.

    Unlike AWT, Swing components are lightweight (not dependent on native peer code) and offer a consistent user experience across operating systems. It remains widely used in legacy enterprise applications, IDEs, and educational tools despite the rise of JavaFX.

    🔧 Core Features of Swing

    ✔ Pluggable L&F
    Metal, Windows, Motif, Nimbus, and custom themes.
    ✔ Rich Components
    JTable, JTree, JSpinner, JEditorPane, JList, and more.
    ✔ MVC Architecture
    Separates data model from visual representation.

    📄 Documented Code Example: Swing Notepad with Code Clock & Syntax Highlighting (Prism.js used for display)

    The following complete Java program implements a functional Swing document editor that includes a real‑time digital clock (code clock), menu bar with save/clear actions, a scrollable text area, and a status bar. This example demonstrates the core principles of Swing: JFrame, JMenuBar, JTextArea, Timer for clock updates, and event handling via lambda expressions. The code block below uses Prism.js library for syntax highlighting, making it ideal for technical documentation.

    📁 SwingNoteWithClock.java — Full working GUI example 
    import javax.swing.*;
import java.awt.*;
import java.awt.event.*;
import java.time.LocalDateTime;
import java.time.format.DateTimeFormatter;

/**
 * Java Swing Document Note with Code Clock
 * Demonstrates JFrame, JTextArea, Timer, Menu, and Event Handling.
 */
public class SwingNoteWithClock {

    private JFrame frame;
    private JTextArea textArea;
    private JLabel clockLabel;
    private JLabel statusLabel;
    private Timer clockTimer;
    private int wordCount = 0;

    public SwingNoteWithClock() {
        // Initialize frame
        frame = new JFrame("📝 Java Swing Document Note | Code Clock");
        frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
        frame.setSize(800, 600);
        frame.setLocationRelativeTo(null);
        frame.setLayout(new BorderLayout());

        // Text area (main document)
        textArea = new JTextArea();
        textArea.setFont(new Font("Monospaced", Font.PLAIN, 14));
        textArea.setLineWrap(true);
        textArea.setWrapStyleWord(true);
        textArea.setBorder(BorderFactory.createEmptyBorder(10, 10, 10, 10));
        JScrollPane scrollPane = new JScrollPane(textArea);
        scrollPane.setBorder(BorderFactory.createTitledBorder("📄 Document Editor"));
        frame.add(scrollPane, BorderLayout.CENTER);

        // Menu bar
        JMenuBar menuBar = new JMenuBar();
        JMenu fileMenu = new JMenu("File");
        JMenuItem saveItem = new JMenuItem("Save Note");
        JMenuItem clearItem = new JMenuItem("Clear All");
        JMenuItem exitItem = new JMenuItem("Exit");

        saveItem.addActionListener(e -> saveDocument());
        clearItem.addActionListener(e -> clearDocument());
        exitItem.addActionListener(e -> System.exit(0));

        fileMenu.add(saveItem);
        fileMenu.add(clearItem);
        fileMenu.addSeparator();
        fileMenu.add(exitItem);
        menuBar.add(fileMenu);

        JMenu editMenu = new JMenu("Edit");
        JMenuItem wordCountItem = new JMenuItem("Show Word Count");
        wordCountItem.addActionListener(e -> showWordCount());
        editMenu.add(wordCountItem);
        menuBar.add(editMenu);

        frame.setJMenuBar(menuBar);

        // Bottom panel: Code Clock + status
        JPanel bottomPanel = new JPanel(new BorderLayout());
        bottomPanel.setBorder(BorderFactory.createEmptyBorder(8, 12, 8, 12));
        
        clockLabel = new JLabel("⏰ Code Clock: --:--:--", SwingConstants.CENTER);
        clockLabel.setFont(new Font("JetBrains Mono", Font.BOLD, 16));
        clockLabel.setForeground(new Color(30, 100, 180));
        
        statusLabel = new JLabel(" Ready | Swing GUI Document Note", SwingConstants.RIGHT);
        statusLabel.setFont(new Font("Segoe UI", Font.PLAIN, 11));
        
        bottomPanel.add(clockLabel, BorderLayout.CENTER);
        bottomPanel.add(statusLabel, BorderLayout.SOUTH);
        frame.add(bottomPanel, BorderLayout.SOUTH);

        // Timer for code clock (updates every second)
        clockTimer = new Timer(1000, e -> updateClock());
        clockTimer.start();

        // Initial status
        updateClock();
        statusLabel.setText(" ✅ Document ready | Type your notes...");
        frame.setVisible(true);
    }

    private void updateClock() {
        LocalDateTime now = LocalDateTime.now();
        DateTimeFormatter formatter = DateTimeFormatter.ofPattern("HH:mm:ss");
        String time = now.format(formatter);
        clockLabel.setText("⚡ CODE CLOCK ⚡   " + time + "   ⚡");
        // optional: update word count dynamically
        String text = textArea.getText();
        wordCount = text.trim().isEmpty() ? 0 : text.trim().split("\\s+").length;
    }

    private void saveDocument() {
        String content = textArea.getText();
        // Simulate saving (could write to file, here just dialog)
        JOptionPane.showMessageDialog(frame, "Document saved (simulated storage).\nCharacters: " + content.length(),
                "Save Note", JOptionPane.INFORMATION_MESSAGE);
        statusLabel.setText("📀 Note saved at " + LocalDateTime.now().format(DateTimeFormatter.ofPattern("HH:mm:ss")));
    }

    private void clearDocument() {
        int confirm = JOptionPane.showConfirmDialog(frame, "Clear all text? This action cannot be undone.",
                "Clear Document", JOptionPane.YES_NO_OPTION);
        if (confirm == JOptionPane.YES_OPTION) {
            textArea.setText("");
            statusLabel.setText("🧹 Document cleared.");
            updateClock();
        }
    }

    private void showWordCount() {
        String text = textArea.getText();
        int words = text.trim().isEmpty() ? 0 : text.trim().split("\\s+").length;
        int chars = text.length();
        int lines = text.split("\n").length;
        JOptionPane.showMessageDialog(frame,
                "📊 Statistics:\n• Words: " + words + "\n• Characters: " + chars + "\n• Lines: " + lines,
                "Document Word Count", JOptionPane.PLAIN_MESSAGE);
    }

    public static void main(String[] args) {
        SwingUtilities.invokeLater(() -> new SwingNoteWithClock());
    }
}

    💡 Code Explanation & Key Swing Concepts:

    • JFrame – Main application window with title, size, and close operation.
    • JTextArea – Multi-line text region, wrapped in JScrollPane for scrolling.
    • Timer – Updates the "Code Clock" label every second, demonstrating real-time UI updates.
    • Menu Bar (JMenuBar, JMenu, JMenuItem) – Provides standard actions: Save, Clear, Exit, and Word Count.
    • Event Handling – Lambda expressions (e -> ...) simplify ActionListener registration.
    • SwingUtilities.invokeLater() – Ensures GUI creation happens on the Event Dispatch Thread (EDT).

    ⏲️ Code Clock Integration in Swing (Detailed Note)

    A "Code Clock" in a Swing application typically refers to a live digital display that shows the current system time, elapsed session time, or a stopwatch. The provided code uses javax.swing.Timer which fires ActionEvent every second. The updateClock() method refreshes the label with the current time. This is the standard way to create real‑time updates without blocking the EDT. You can also attach a Chronometer to measure code execution performance, but the simplest form is a digital clock.

    📌 Key takeaway: Swing’s Timer (javax.swing.Timer) is ideal for periodic UI updates. Unlike java.util.Timer, its actions execute on the EDT, ensuring thread safety when updating Swing components.
